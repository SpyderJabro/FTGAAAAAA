SpamAll (Модуль) Модуль для *массовой* рассылки сообщений пользователям чата в котором Вы пропишете команду.

Стоимость: 300₽ После покупки модуль будет доступен для загрузки в разделе «Мои покупки».
SpamAll (Клиент) *Клиент* переписанный с одноименного модуля, используется для быстрой рассылки пользователям без установки FTG.

Стоимость: 600₽ После покупки модуль будет доступен для загрузки в разделе «Мои покупки».
TimeMessage Модуль для плановой отправки сообщения в указанные чаты с указанным интервалом. *Поддерживает приватные чаты и каналы.*

Стоимость: 230₽ После покупки модуль будет доступен для загрузки в разделе «Мои покупки».
